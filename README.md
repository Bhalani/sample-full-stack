# Calorie Tracking App


## Technology

```
Ruby 2.7.6,
Rails 7.0.2
vite-rails 3.0.8 (gem for frontenv compiler)
Vue 3.2.33
pinia 2.0.13 (Vue Store management library)
yarn 1.22.17
```


## Installation

```
bundle install
yarn install
bundle exec rails db:create
bundle exec rails db:migrate
bundle exec rails db:seed
```

## Booting the app

```
foreman start
```
